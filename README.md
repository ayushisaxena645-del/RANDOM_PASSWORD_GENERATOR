Random Password Generator 🔐

Overview

This project is a Random Password Generator developed using HTML, CSS, and JavaScript. It helps users generate strong and secure passwords instantly.

Features

- Generate random passwords
- Select password length
- Include uppercase and lowercase letters
- Include numbers and special characters
- Simple and user-friendly interface
- Responsive design

Technologies Used

- HTML
- CSS
- JavaScript

How It Works

1. Open the application in a browser.
2. Select password requirements.
3. Click on the Generate Password button.
4. Copy and use the generated password.

Project Objective

The objective of this project is to create a secure and easy-to-use password generator that helps users create stronger passwords.

Future Improvements

- Add password strength meter
- Add copy-to-clipboard feature
- Save password history
